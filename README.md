# 구현된 기능 <br/>
* Asset Store에 있는 조이스틱을 다운받아서 적용
* 키보드로도 조작할 수 있도록 GetAxis 구현 후 주석처리
* 먹은 코인 수, 시간, 베스트시간, Restart 구현
* 코인이 제자리에서 빙글빙글 도는것 구현
